    Create MetaMask/Trust wallet

    Create a MetaMask Wallet using either a desktop computer or an TrustWallet using Android Mobile Device. That will allow you to send, and receive $FREE

    Send BNB to your wallet

    You can transfer it to your MetaMask/Trust Wallet from exchanges like Pintu, Indodax, TokoCrypto, Binance, etc. Make sure to use the BEP-20 Network when transferring BNB.
